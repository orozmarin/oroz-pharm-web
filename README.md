# Oroz PHARM

Web stranica za [Oroz PHARM](https://oroz-pharm.hr) — poljoprivrednu ljekarnu koja posluje od 1998. godine u Pleternici i Požegi, Hrvatska.

Katalog od 12.000+ proizvoda: sjemena, gnojiva, sredstva za zaštitu bilja, stočna hrana, oprema za navodnjavanje i više. Uz katalog, stranica nudi edukativni blog i pregled lokacija prodavaonica.

## Tech Stack

Next.js 15 · Payload CMS v3 · PostgreSQL · TypeScript · Tailwind CSS v4
